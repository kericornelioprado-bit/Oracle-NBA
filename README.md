# Oráculo NBA 🏀

Modelo Predictivo para Apuestas de Valor (Value Betting) en la NBA.

## Estructura del Proyecto
- `data/`: Datasets (Raw, Processed).
- `src/`: Scripts de extracción y modelado.
- `notebooks/`: Análisis Exploratorio (EDA).
- `infra/`: Terraform para GCP.
- `tests/`: Pruebas de calidad.

## Configuración del Entorno
1. Construir la imagen: `docker build -t oracle-nba .`
2. Ejecutar contenedor: `docker run -it oracle-nba`
